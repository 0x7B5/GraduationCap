# GraduationCap

The script I ran on my Raspberry Pi for my graduation cap. 

## What I used
1. Raspberry Pi Zero (With Headers) 
2. RGB LED Matrix Panel - 32x32
3. Breadboard Jumper Wires
4. 10,000 mAh Portable Charger
5. 2x Micro-USB Cables
